## 포함 플러그인

### Strategy Condition (조건 전략)
- RSI: RSI 과매수/과매도 조건
- MACD: MACD 크로스오버 조건
- BollingerBands: 볼린저밴드 조건
- VolumeSpike: 거래량 급증 조건
- ProfitTarget: 익절 조건
- StopLoss: 손절 조건

### New Order (신규 주문)
- MarketOrder: 시장가 주문
- LimitOrder: 지정가 주문

### Modify Order (정정 주문)
- TrackingPriceModifier: 가격 추적 정정

### Cancel Order (취소 주문)
- TimeStopCanceller: 시간 초과 취소
