# 76 - 골든크로스 자동 매수 (MA5/MA20, 선물 모의)

> **시나리오**: "이동평균선 골든크로스 나오는 종목 있으면 알아서 매수해줘."

## 흐름

```
Schedule(일 1회) ─┬─► Watchlist → Historical → MACross(golden) ─┐
                  │                                              ▼
                  └─► Account ─────────────────► SymbolFilter (미보유만)
                                                        │
                                                   Buy → Telegram
```

## 골든크로스란?

- **단기 이동평균** (MA5: 최근 5일 평균) 이 **장기 이동평균** (MA20: 최근 20일 평균) 을 **위로 돌파**
- 전일까지는 MA5 ≤ MA20, 오늘 MA5 > MA20 → 골든크로스 발생
- **해석**: 단기 추세가 장기 추세보다 강해짐 → 상승 추세 전환 초입 가능성

데드크로스는 반대 (MA5 가 MA20 을 아래로 돌파 → 하락 추세).

## MovingAverageCross 플러그인

```json
"fields": {
  "short_period": 5,
  "long_period": 20,
  "cross_type": "golden"
}
```

| cross_type | 감지 대상 |
|-----------|----------|
| `"golden"` | 단기 MA 가 장기 MA 를 상향 돌파 (매수 신호) |
| `"dead"` | 단기 MA 가 장기 MA 를 하향 돌파 (매도 신호) |

## MA 기간 선택 가이드

| 성격 | short/long | 특징 |
|------|-----------|------|
| 초단기 | MA3 / MA10 | 많은 신호, 잦은 가짜 신호 |
| **단기 (기본)** | MA5 / MA20 | 균형 |
| 중기 | MA10 / MA50 | 주간 스윙 트레이딩 |
| 장기 | MA50 / MA200 | "Golden/Death Cross" 클래식 |

## 체크 주기

```json
"cron": "0 10 * * 1-5"
```

평일 10:00 HKT — 장 시작 후 30분 경과 (변동성 안정화) 지점에서 전일 종가 기준 크로스 감지.

## 안전장치

1. **SymbolFilter(difference)** — 이미 보유 중인 종목은 스킵 (중복 매수 방지)
2. **resilience.fallback: skip** — 일부 주문 실패 시 다음 종목 계속
3. **paper_trading: true** — 모의투자

## 확장 아이디어

- **데드크로스 청산 추가**: 같은 워크플로우에 `cross_type: "dead"` 브랜치 → 매도
- **다중 확인**: MA 크로스 + RSI 40 이상 → LogicNode AND 게이트
- **트레일링 스탑**: 매수 후 #74 워크플로우와 결합하여 자동 손절
- **변동성 필터**: ATR 기반 노이즈 필터 추가

## 주의사항

- MA 크로스는 **후행 지표** — 이미 추세가 진행 중인 후 신호 발생
- **횡보장에서 약점**: 가짜 신호 빈번
- **강한 추세장에서 최강**: 상승 추세 초기 포착에 유리
